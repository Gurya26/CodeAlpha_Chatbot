# 🤖 AI Chatbot (CodeAlpha Internship)

This project is a FAQ-based AI chatbot built using Flask and Machine Learning techniques.

## 🚀 Features

* TF-IDF Vectorization
* Cosine Similarity Matching
* Real-time Chat UI
* Predefined FAQ dataset

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-learn
* HTML, CSS

## 🔗 Live Demo

[https://chatbot-website-5.onrender.com/](https://codealpha-chatbot-iydj.onrender.com/)

## 📌 How it Works

* User inputs a query
* System converts it into TF-IDF vector
* Cosine similarity is used to find the best match
* Relevant answer is returned

## 📁 Project Structure

* app.py → Backend logic
* faq.csv → Dataset
* templates/ → Frontend HTML
* static/ → CSS

## ▶️ Run Locally

1. Clone the repository
git clone https://github.com/your-username/repo-name
