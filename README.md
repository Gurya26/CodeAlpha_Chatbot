# 🤖 AI Chatbot (CodeAlpha Internship)

This project is a FAQ-based AI chatbot built using Flask and Machine Learning techniques.

## 🚀 Features

* TF-IDF Vectorization
* Cosine Similarity Matching
* Real-time Chat UI
* Predefined FAQ dataset

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-learn
* HTML, CSS

## 🔗 Live Demo

https://codealpha-chatbot-iydj.onrender.com/

## 📌 How it Works

* User inputs a query
* System converts it into TF-IDF vector
* Cosine similarity is used to find the best match
* Relevant answer is returned

## 📁 Project Structure

* app.py → Backend logic
* faq.csv → Dataset
* templates/ → Frontend HTML
* static/ → CSS

## ▶️ Run Locally

1. Clone the repository
git clone https://github.com/Gurya26/CodeAlpha_Chatbot

2. Install dependencies
pip install -r requirements.txt

3. Run the app
python app.py

4. Open in browser
http://127.0.0.1:5000/
